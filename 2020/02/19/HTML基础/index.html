HTML基础

  • html基本结构：html文件<html></html>,头部信息,网页主体内容<body></body>
  • 水平线：<hr/>（单标签）
  • 段落标签：<p></p>
  • 元素：开始标签+内容+结束
  • 颜色：bgcolor = “red”，开始标签后面添加标签属性，属性与属性之间用空格隔开
  • 注释：<!--注释内容-->
  • 声明doctype类型，放在html文档第一行，<!DOCTYPE html> (不是标签)
  • 解决乱码在<head></head>标签之间添加：<meta http-equiv ="Content" content = "text/html; charse = utf-8"/>
  • 标题标签<h1></h1>~<h6></h6>
  • 段落标签 <p></p>
  • 换行标签 <br/>
  • 对齐属性 ： left，right， center， justify(对行进行延伸，这样每行都有相同的长度)。
  • 换行标签，<br/>，可以使该标签后面的换行
  • 空格 &nbsp;
  • 保留编辑器中的文本格式：<pre></pre>
  • 水平线 <hr/>， 属性：width（设置像素或者百分比），color, align, noshade(设置水平线无阴影)
  • 修饰标签: 文字斜体<i></i>、<em></em>,加粗<b></b>、<strong></strong>， 下标<sub></sub>， 上标 <sup></sup >, 下划线<ins>, 删除线 <del>
  • 特殊符号：< &lt;,>&gt, 已注册&reg,版权&copy, 商标&trade, 不断行空白&nbsp
  • 无序列表：<ul></ul>,中间内容:<li></li>,type属性设置列表符号形状:disc,square,circle
  • 有序列表：<ol></ol>，中间列表项：<li></li>，type属性值1，a, A, i(小写罗马数字), I（大写罗马数字）
  • 定义列表：<dl></dl>,定义列表项：<dt></dt>,定义列表描述：<dd></dd>(定义列表内可以多个定义列表项，每个定义列表项中可以多个定义列表项描述）
  • 图案标签（单标签）：<img src = " " alt = " "..../>
  • img属性： src值为URL,alt值为文字（无法正常显示图像的时候进行文字的显示），height值为数值和百分比， width值为数值和百分比；
  • HTML路径：相对路径，绝对路径
  • 超链接标签：<a href = " ">内容</a>
  • 站内链接：相对路径； 站外链接：绝对路径；
  • 空链接（链接效果但不确定链接目标）：href设置为#，点击不会产生跳转
  • 链接属性：target链接的目标窗口，默认为当前页面，_self: 当前页面，_blank:创建新的窗口打开页面，_top, _parent；title链接的提示文字； name链接的命名;
  • 定义锚(跳转相应起始点位置)：<a href = "#锚名1">目录1</a>,<a href = "..." name =“锚名1”>内容</a>;
  • 不同页面定义锚：网页1 <a href = “网页名称#锚名”>...</a>;网页2 <a name = "锚名"> ... </a >;
  • 邮件链接：<a href = "mailto:邮件地址">...</a>
  • 文件下载： <a href = "下载文件的地址">...</a>
